# espadaNaForja.AppleAcademy
Este projeto é a resolução do Desafio Final (lesson 05 - challenge) da Apple Academy | Centro Universitario Senac Santo Amaro. Ele consiste em um sistema interativo simples em swift, ambiente em um universo de fantasia RPG.

Proposta do Projeto

O sistema implementa uma Loja de Forja de Espadas simples , onde o aventureiro pode interage com ferreiro João para comprar equipamentos.

Conceito Utilizados
- Coleções 
- Funções
- Controle de Fluxo
- Tema RPG/fantasia
